# Delete Favorites

**Method** : `DELETE`

**URL** : `https://galaxy-media.2.us-1.fl0.io/api/favorites/delete/{favId}`

**Auth required** : `True`

**Lang Support**: `False`

**Parameters** : 
* `favId`: You need to provide a valid ID belonging to a Favorite.

<br />

## Example of a Successfull Response

**Code** : `204 NO CONTENT`
